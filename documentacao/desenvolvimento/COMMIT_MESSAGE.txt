feat: Sistema completo de referências culturais brasileiras

Implementação de banco de dados e sistema de scraping para enriquecer
a IA do FlertAI com gírias, memes e referências culturais autênticas
do Brasil.

## 🎯 Funcionalidades Adicionadas

### Database (PostgreSQL/Supabase)
- Nova tabela `cultural_references` com schema completo
- 6 índices otimizados (incluindo full-text search PT-BR)
- 3 funções SQL helper:
  * get_random_cultural_reference() - Busca aleatória por tipo/região
  * search_cultural_references() - Busca textual com relevância
  * get_cultural_references_stats() - Estatísticas agregadas
- 4 RLS policies configuradas (read público, write autenticado)
- 25 registros seed embutidos na migration

### Scripts Python
- Sistema modular de web scraping (scrapy/beautifulsoup4)
- 87 referências culturais curadas manualmente
- Script de inserção com batch processing
- Validação e limpeza automática de dados
- Suporte para 12 tipos e 6 regiões do Brasil

### Documentação
- README completo com guias de uso (450 linhas)
- Setup passo a passo com checklist (370 linhas)
- Integração com Edge Function documentada (480 linhas)
- 4 guias adicionais (INICIO_RAPIDO, STATUS_FINAL, etc)

## 🚀 Como Usar

1. Aplicar migration SQL no Supabase Dashboard
2. Configurar Service Role Key no .env
3. Executar: `python scripts/scraper/run_after_config.py`
4. Verificar 87 referências inseridas

Consulte: INICIO_RAPIDO.md para guia de 5 minutos
